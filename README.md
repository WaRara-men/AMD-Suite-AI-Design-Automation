# 🤖 AMD Suite v8.0 - Robot Actuator Selection Center

**「勘」で部品を買うのは、もう終わりにしましょう。**

AMD Suite は、ロボット制作における最も過酷な意思決定――**「どのモーターを買えばいいのか？」**をAIと物理シミュレーションで自動化する、ロボメカ学科生のための最強設計スイートです。

## 🚀 なぜこのプロダクトが必要なのか？ (The Problem)
ロボット制作の現場では、多くの学生が以下の問題に直面します：
- **計算ミス:** 「アームの腕を作ったが、重すぎてモーターが回らなかった」
- **迷い:** 「ミスミやAmazonのサイトを数時間彷徨っても、どれが最適か選べない」
- **予算:** 「高いパーツを買ったのに、実はもっと安くて軽い選択肢があった」

**AMD Suite は、これらの「迷い」と「失敗」をゼロにします。**

---

## 🌟 主な機能 (Core Features)

### 1. ⚙️ 物理演算ベースのモーター選定
「持ち上げたい重さ」と「アームの長さ」を入力するだけで、AIが裏側でトルク計算（$T = m \cdot g \cdot L$）を実行。
カタログデータベース（CSV）から、**「パワーが足りて、かつ最も軽量で安価なモーター」**を一瞬で選び出します。

### 2. 📊 リアルタイム・トレードオフ分析
ダッシュボード上のグラフで、カタログ内の全モーターと比較。
「なぜそのモーターが選ばれたのか」を視覚的に納得した上で設計を進めることができます。

### 3. 📜 ロボット選定仕様証明書 (PDF)
選定結果を「公式な技術ドキュメント」として自動発行。
必要な理論トルク、選定された型番、価格、納期、安全性が1枚のPDFに集約されます。これはあなたの研究の「正当性」を証明する強力なエビデンスになります。

### 4. 📢 バイリンガル音声フィードバック
解析完了を、日本語と英語の流暢なボイスで報告。忙しい制作の現場をAIがナビゲートします。

## 🖥️ 使いかた (Quick Start)

1. **アプリ起動**: MATLABで `src/AMD_App` を実行。
2. **仕様入力**: スライダーで「Payload（重さ）」と「Arm Length（長さ）」を調整。
3. **証明書発行**: **[🚀 SELECT MOTOR]** ボタンを押すだけで、PDFが生成され、目の前で開きます。

---

## 📂 プロジェクト構成
- **`src/`**: ロボット設計アプリ本体 & 物理演算エンジン
- **`data/`**: モーターカタログ・データベース (`Standard_Parts_Catalog.csv`)
- **`out/`**: 生成された選定証明書 (PDF)
