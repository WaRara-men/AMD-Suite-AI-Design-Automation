# 🤖 AMD Suite (Algo-Mech Designer Suite) v23.4

> **"From Physics to Procurement in Seconds."**  
> (物理演算から購入までを数秒で。)

---

## 🚀 プロジェクト概要 (Vision)
ロボット制作における「どのモーターや部品を選べばいいのか？」という迷いと、計算ミスによる「作ってみたが動かない」という悲劇を撲滅するために開発されました。  
特に、アームの自重を無視した設計や予算外のパーツ選定といった、学生エンジニアが陥りがちな罠を物理シミュレーションとAIデータベースで解決します。

## 🧠 死守すべき 5 大機能 (Core Features)

### 1. 全能の 5 モード・物理演算エンジン
あらゆるロボット機構をカバーする解析モードを搭載：
- 🦾 **Arm (腕)**: アームの半径から自重を計算し、先端の錘（Payload）を含めた旋回トルクを算出。
- 🏗️ **Lift (昇降)**: 滑車効率（Efficiency）を考慮した垂直巻上力の詳細解析。
- 🏎️ **Mobile (足回り)**: 坂道の角度（Incline）と路面摩擦を考慮した走行駆動力を計算。
- 🔋 **Power (電源)**: 消費電流と稼働時間から最適な Li-Po バッテリーを自動選定。
- 🔩 **Bolt (結合)**: 荷重を分担するネジの本数から、ボルトのせん断・破断強度をシミュレーション。

### 2. 💎 超解像 3D ライブプレビュー
設計中のパラメータをリアルタイムで可視化。
- 「オレンジの錘が付いたアーム」
- 「青い滑車と昇降ボックス」
- 「斜面を登るタイヤ」
など、直感的な 3D モデルを描画し、物理的なイメージのズレを防ぎます。

### 3. 📜 プロ仕様の「技術鑑定証明書 (PDF)」自動発行
選定の論理的根拠を日本語と英語の両方で詳述したリッチなドキュメント。
- アプリ画面の 3D スナップショットを埋め込み。
- 解析グラフを自動生成・配置。
- 教授やチームへの提出、または自分の設計の正当性証明に。

### 4. 🛒 実売データ ＆ クリック購入リンク
Amazon や DigiKey 等に実在する製品（MG996R, Dynamixel 等）をデータベース化。
- 選定されたパーツは PDF 内のリンクから直接商品ページへアクセス可能。
- 計算から調達までのリードタイムを最小化。

### 5. 🎙️ 多言語ストーリーテリング・ボイス
解析結果を日本語と英語の流暢なフルボイスで報告。
エンジニアとしての達成感を最大化する音声演出で、無機質な計算に生命を吹き込みます。

## 🛠️ 開発のこだわり (Engineering Principles)

### 不滅のモジュール構造 (Modular Design)
`Logic`, `Visuals`, `Report`, `Voice`, `App` の各機能を完全分離。
新機能の追加が既存のシステムを壊さない「不沈艦」の設計思想を採用しています。

### 徹底したユーザーフレンドリー (UX Strategy)
- **High-Visibility UI**: 暗いラボでも読みやすい高輝度ホワイト文字を採用。
- **Japanese Guides**: すべてのパラメータに日本語のヒントガイドを搭載。

---

## 🖥️ はじめかた (Getting Started)

1. **アプリ起動**: MATLAB で `AMD_App.m` を実行します。
2. **設計入力**: スライダーや数値入力で仕様（重さ、長さ、角度等）を調整。
3. **解析 & 発行**: `[🚀 ANALYZE & CERTIFY]` ボタンをクリック。音声レポートが流れ、数秒で PDF 証明書が自動生成されます。
