# 🚀 AMD Suite v7.4 - The Ultimate Standalone AI Engineering Desktop

**AI設計、リアルタイム3D、公式証明書発行。すべてをMATLAB一つで完結させる。**

Algo-Mech Designer (AMD) Suite は、工学設計の煩雑なプロセスをAIで自動化し、設計者に「思考」の時間を取り戻させるためのプロフェッショナル・スイートです。

## 🌟 主要機能 (Core Features)

1. **🧠 超高速 AI 最適化エンジン**: 
   目標荷重・予算上限・安全率をリアルタイムに解析し、鉄・アルミ・カーボンの中から最も「軽量」かつ「低コスト」な最適解を瞬時に算出します。
2. **💎 Hyper-Virtual 3D プレビュー**: 
   外部CADソフト不要。MATLAB上で光沢や質感を再現したリアルタイム・レンダリングを行い、設計した形状をマウスで自由自在に確認できます。
3. **📜 公式設計検証証明書 (Design Certificate)**: 
   ボタン一つで、選定の論理的根拠、詳細スペック、偽造防止IDを含んだプロ仕様の「技術証明書 (PDF)」を自動生成・自動表示します。
4. **🔊 多言語スマート・ボイス案内**: 
   解析の状況や結果を、日本語と英語の両方で流暢に音声報告。グローバルな開発環境に対応しています。
5. **🧹 セルフ・クリーニング・システム**: 
   一時的なゴミファイルを自動で整理し、常にクリーンなプロジェクト環境を維持します。

## 🖥️ 使いかた (Quick Start)

### 1. 起動
MATLABのコマンドウィンドウで以下を実行してください：
```matlab
cd src
AMD_App
```

### 2. 操作
- スライダーを動かして設計パラメータを調整。
- 右上のボタンで言語（JP/EN）を切替。
- **[🚀 GENERATE DESIGN CERTIFICATE]** を押すと、PDF証明書が発行され、自動的に最前面で開きます。

---

## 📂 プロジェクト構成
- **`src/`**: アプリケーション本体・AIエンジン (`AMD_App.m`, `AMD_Main_Brain.m`)
- **`data/`**: 素材・価格の市場データベース (`Standard_Parts_Catalog.csv`)
- **`out/`**: 生成されたすべての証明書 (PDF) と解析ログ
