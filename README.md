## 🧐 AMD Suite とは？ (What is this?)
AMD Suite (Algo-Mech Designer) は、**「エンジニアの思考を止めてはいけない」**という理念から生まれた、次世代の設計自動化ツールです。

通常、材料選定やコスト計算、3Dモデルの作成、そして証明書の発行には複数のソフトウェアと多大な時間が必要です。AMD Suite はこれらすべてのプロセスを AI が肩代わりし、MATLAB 上の直感的なダッシュボード一つで完結させます。

### 💡 なぜ AMD Suite なのか？
- **迷わない**: 最適な材料（鉄・アルミ・カーボン）とコストを AI が瞬時に提案。
- **作らない**: CADソフトを立ち上げることなく、高品位な 3D プレビューをその場で確認。
- **待たない**: 解析完了と同時に、プロ仕様の技術証明書 (PDF) を自動生成。

---

## 🌟 主要機能 (Core Features)

1. **🧠 超高速 AI 最適化エンジン**: 
   目標荷重・予算上限・安全率をリアルタイムに解析。瞬時に最も「軽量」かつ「低コスト」な解を算出。
2. **💎 Hyper-Virtual 3D プレビュー**: 
   MATLAB 上で光沢や質感を再現したリアルタイム・レンダリング。マウス操作で自由自在に全方位確認。
3. **📜 公式設計検証証明書 (Design Certificate)**: 
   論理的根拠、詳細スペック、偽造防止IDを含む PDF 証明書をワンボタンで発行・自動表示。
4. **🔊 多言語スマート・ボイス案内**: 
   進捗や結果を日英 2 言語で音声報告。グローバルな開発環境をサポート。
5. **🧹 セルフ・クリーニング・システム**: 
   一時ファイルを自動整理し、常にクリーンなプロジェクト環境を維持。

## 🖥️ 3分でクイックスタート (Quick Start)

### ⚙️ 前提条件
- **MATLAB** がインストールされていること。（推奨：R2023a以降）

### 🚀 起動手順
1. このリポジトリをダウンロード（または Git Clone）。
2. MATLAB のコマンドウィンドウで以下を実行してください：
```matlab
cd src
AMD_App
```

### 🎮 操作はかんたん
- **調整**: スライダーを動かすだけで、AI が背後で数千回のシミュレーションを行います。
- **発行**: **[🚀 GENERATE DESIGN CERTIFICATE]** を押せば、即座に証明書が画面に表示されます。

---

## 📂 プロジェクト構成
- **`src/`**: アプリケーション本体・AIエンジン (`AMD_App.m`, `AMD_Main_Brain.m`)
- **`data/`**: 素材・価格の市場データベース (`Standard_Parts_Catalog.csv`)
- **`out/`**: 生成されたすべての証明書 (PDF) と解析ログ
