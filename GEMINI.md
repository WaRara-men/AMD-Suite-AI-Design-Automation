# 🤖 AMD Suite (Algo-Mech Designer Suite) v23.5 - Project Milestone

## 🚀 Status: COMPLETE (v23.5)
The platform is fully integrated with 5 design modes and a high-fidelity reporting engine.

### 🧠 Core Modules (Physics Engines)
1.  **🦾 Arm**: Analyzes torque with self-weight (aluminum density 0.0000027 kg/mm³).
2.  **🏗️ Lift**: Pulley efficiency and vertical lift analysis.
3.  **🏎️ Mobile**: Incline driving force calculation with road friction.
4.  **🔋 Power**: Li-Po battery capacity and run-time simulation.
5.  **🔩 Bolt**: Structural shear/fracture strength simulation for bolt arrays.

### 💎 Key Features
- **3D Live Preview**: Real-time visualization of robot mechanics within the MATLAB UI.
- **Eloquent PDF Certificate**: Professional-grade reports with 3D snapshots, ranking graphs, and direct purchase links to Amazon/DigiKey.
- **Multi-Lingual Voice (AI Narrator)**: Fluent Japanese/English reporting via .NET `System.Speech`.

### 🛠️ Technical Stack
- **Engine**: MATLAB R2024b (App Designer, Statistics and Machine Learning Toolbox).
- **Reporting**: MS Word (via `actxserver`) for high-resolution PDF export.
- **Voice**: .NET Framework (Windows Speech Synthesis).
- **Data**: CSV-based catalogs for motors, batteries, and bolts.
