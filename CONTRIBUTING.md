# Contributing to AMD Suite

First off, thank you for considering contributing to AMD Suite! It's people like you that make AMD Suite such a great tool.

AMD Suite への貢献を検討していただき、ありがとうございます！あなたのような方々の協力が、このツールをより良くします。

## 🌐 Bilingual Policy / バイリンガル方針
This project is bilingual (English & Japanese). Please provide descriptions in both languages if possible.
本プロジェクトは日英バイリンガルです。可能であれば、両方の言語で説明を記載してください。

## 🛠️ How to Contribute / 貢献方法
1. **Fork** the repository.
2. Create a **Branch** (`git checkout -b feature/amazing-feature`).
3. **Commit** your changes (`git commit -m 'Add some amazing feature'`).
4. **Push** to the Branch (`git push origin feature/amazing-feature`).
5. Open a **Pull Request**.

## 📝 Coding Standards / コーディング規格
- Follow MATLAB best practices.
- Ensure automated reports and export CSVs remain compatible.
- MATLABのベストプラクティスに従ってください。
- 自動レポート生成やCSV出力の互換性を維持してください。
